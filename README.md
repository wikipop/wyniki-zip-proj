# next-template

A Next.js 13 app build with Radix UI and Tailwind CSS.

## Start up

```bash
npm install
npx run dev
```

## Features

- Full responsive
- Build on Next.js 13 with app directory structure
- Full ts typing
- Radix UI
- Tailwind CSS
- Dark mode with `next-themes`
- Tailwind CSS class sorting, merging and linting.
