# next-template

A Next.js 13 app build with Radix UI and Tailwind CSS.

## Start up

```bash
npm install
npx run dev
```

## Features

- Radix UI Primitives
- Tailwind CSS
- Fonts with `next/font`
- Icons from [Lucide](https://lucide.dev)
- Dark mode with `next-themes`
- Tailwind CSS class sorting, merging and linting.
